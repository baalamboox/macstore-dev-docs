CKEditor es un editor WYSIWYG en línea que se utiliza para editar documentos HTML (o sus fragmentos) en el navegador.  
El contenido web se crea en HTML, un lenguaje de marcado que incluye texto y etiquetas especiales que describen este texto y le añaden significado semántico.

## Cómo agregarlo a la página
Para comenzar, crea una página HTML sencilla con un "textarea" elemento en ella. Luego tendrás que hacer dos cosas:

Incluya el "script" elemento de carga CKEditor 4 en su página.
Utilice el CKEDITOR.replace()método para reemplazar el "textarea" elemento existente con CKEditor 4.
<hr />
 <Card title="Ejemplo" icon="setting">
```html
   <html lang="en">
    <head>
        <meta charset="utf-8">
        <script src="../vendor/ckeditor/ckeditor.js"></script>
    </head>
    <body>
        <form>
            <textarea name="editor1" id="editor1" rows="10" cols="80">
            </textarea>
            <script>
                CKEDITOR.replace( 'editor1' );
            </script>
        </form>
    </body>
</html>
```
<Aside>El orden para agregar los scrip debe ser el mismo que se muestra en el ejemplo</Aside>
