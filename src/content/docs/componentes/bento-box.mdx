Los bento grids son diseños que dividen el contenido en áreas distintas, creando una presentación visualmente atractiva y organizada.
Facilita la segmentación del contenido, lo que mejora la accesibilidad y la navegación del usuario.

## Implementación
<hr />

Para realizar la Implementación de los Bento Box se tiene que tener en cuenta el uso de los siguientes archivos de css
para su correcto funcionamiento.

:::caution[Importante]
    <FileTree>
        - public
            - css
                - root.css
                - buttons.css
                - bento-home.css
        - index.html
    </FileTree>
    
    En el archivo **index.html** o donde se desea usar los Bento Box se debe primero cargar el
    archivo **root.css**, **buttons.css** y finalmente el archivo **bento-home.css**.

    ```html
    <!DOCTYPE>
    <html>
        <head>
            <link rel="stylesheet" href="./public/css/root.css" />
            <link rel="stylesheet" href="./public/css/buttons.css" />
            <link rel="stylesheet" href="./public/css/bento-home.css" />
        </head>
    </html>
    ```
:::
