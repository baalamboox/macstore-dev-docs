Los botones son elementos de interfaz de usuario que permiten a los usuarios interactuar con una aplicación o sitio web. Se utilizan para realizar acciones específicas, como iniciar sesión, agregar un elemento a un carrito de compras o proporcionar información adicional.

## Uso
<hr />

El uso de los botones es sumamente fácil, solo hay que tener en cuenta algunos detalles de configuración de archivos como es el caso de las fuentes y los colores ya que los botones hacen uso de estos archivos:

- **fonts.css**
- **root.css**

:::caution[Ejemplo de uso]
    Ejemplo de directorios <small><b>(Puede cambiar)</b></small>
    <FileTree>
        - public
            - css
                - root.css
                - fonts.css
                - buttons.css
        - index.html
    </FileTree>

    En el archivo **index.html** o donde se desean usar los botones se debe primero cargar el
    archivo **root.css** y después **fonts.css**.

    ```html
        <!DOCTYPE>
        <html>
            <head>
                <link rel="stylesheet" href="/public/css/root.css" />
                <link rel="stylesheet" href="/public/css/fonts.css" />
                <link rel="stylesheet" href="/public/css/buttons.css" />
            </head>
        </html>
    ```

    **Nota:** Es recomendable hacer uso de layouts para evitar llamar más de una vez el mismo archivo.
:::
