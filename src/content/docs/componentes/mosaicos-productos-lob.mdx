Los LOB's (Line of business) es un tipo de lista sobre los productos

## Implementación
<hr />

Para realizar la implementación de los LOB's se tiene que tener en cuenta el uso de los siguientes archivos de css
para su correcto funcionamiento.

:::caution[Importante]
    <FileTree>
        - public
            - css
                - root.css
                - color.css
                - buttons.css
                - lob-tiles.css
        - index.html
    </FileTree>
    
    En el archivo **index.html** o donde se desea usar los LOB's se debe primero cargar el
    archivo **root.css**, **color.css**, **buttons.css** y finalmente el archivo **lob-tiles.css**.

    ```html
    <!DOCTYPE>
    <html>
        <head>
            <link rel="stylesheet" href="./public/css/root.css" />
            <link rel="stylesheet" href="./public/css/color.css" />
            <link rel="stylesheet" href="./public/css/buttons.css" />
            <link rel="stylesheet" href="./public/css/lob-tiles.css" />
        </head>
    </html>
    ```
:::
