San Francisco Pro es una tipografía diseñada por Apple, hace que los texto sean consistentes, legibles
y amigables asegurando una óptima legibilidad sin importar el tamaño y la resolución de pantalla.

## Clasificación
La tipografía San Francisco se divide en 3 familias que son las siguientes:
- Display: Se usa típicamente para títulos.
- Rounded: Se usa para texto o título donde las letras en sus extremos sean redondeadas.
- Text: Se usa típicamente para textos.

## Uso
:::caution[¡No olvides!]
    Para poder implementar las tipografías se tiene que hacer uso del archivo CSS llamado **fonts.css** y colocarlo
    en el archivo raíz o en donde se desee usar. Ejemplo:
    <FileTree>
        - public
            - css
                - fonts.css
        - index.astro
    </FileTree>
    Ya solo queda enlazar el archivo **fonts.css** mediante la etiqueta **link**.
    ```html
    <link rel="stylesheet" href="./css/fonts.css" />
    ```
:::

Para implementar las tipografías San Francisco se tiene que hacer uso del atributo **class** de los elementos de texto
en los que se desea utilizar, teniendo en cuenta la siguiente nomenclatura:
- sfpd-*: Si se desea usar alguna tipografía de la familia Display.
:::note[Ejemplo:]
```html
<p class="sfpd-bold">Texto Display Bold</p>
```
<p class="sfpd-bold">Texto Display Bold</p>
:::
- sfpt-*: Si se desea usar alguna tipografía de la familia Text.
:::note[Ejemplo:]
```html
<p class="sfpt-bold">Texto Text Bold</p>
```
<p class="sfpt-bold">Texto Text Bold</p>
:::
- sfpr-*: Si se desea usar alguna tipografía de la familia Rounded.
:::note[Ejemplo:]
```html
<p class="sfpr-bold">Texto Rounded Bold</p>
```
<p class="sfpr-bold">Texto Rounded Bold</p>
:::

## Usando CSS
Para usar las tipografías sin uso de clases, solo es necesario hacer uso de la propiedad **font-family: valor;** y poner como valor el tipo de fuente que
se desea usar.
:::note[Ejemplo:]
```css
h1 {
    font-family: 'sf-pro-display-regular';
}

p {
    font-family: 'sf-pro-text-light';
}

.mi-texto {
    font-family: 'sf-pro-rounded-ultralight';
}
```
