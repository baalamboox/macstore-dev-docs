Bootstrap es un librería de desarrollo web gratuito y de código abierto. Su propósito principal es facilitar la creación de sitios web responsivos
y adaptables a diferentes dispositivos, utilizando una combinación de HTML, CSS y JavaScript.

La estructura anterior muestra los diferente archivos que conforman Bootstrap modificado para CX MacStore,
a continuación de enlista una breve descripción de cada archivo:

- **css/**
    - `alert.css:` Contiene todos los estilos necesarios para el correcto funcionamiento de las alertas.
    - `animations.css:` Aquí se encuentran todas las animaciones que requiere Bootstrap.
    - `modals.css:` Aquí se encuentran todas las animaciones que requiere Bootstrap.
- **js/**
    - `bootstrap.min.js:` Contiene toda la funcionalidad de la librería de Bootstrap, para hacer funcionar por ejemplo la apertura y cierre de los collapse.
