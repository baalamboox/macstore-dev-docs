Los modales de Bootstrap son ventanas emergentes que se utilizan para mostrar información adicional o para realizar acciones específicas sin redirigir a otra página.

## Implementación

La Implementación de los modales de Bootstrap adaptados a CX MacStore es sumamente sencillo, solo hay que tener en cuenta
algunos archivos que se requieren para su correcto funcionamiento.

:::caution[Uso]

:::
